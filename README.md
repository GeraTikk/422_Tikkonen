# Учебная практика - Разработка WPF приложения

## Описание
WPF приложение для управления платежами с системой авторизации, регистрации и ролевой моделью (Администратор/Пользователь).

## Основной функционал
- **Авторизация** с капчей после 3 неудачных попыток
- **Регистрация** пользователей с валидацией пароля
- **Ролевой доступ**: Администратор и Пользователь
- **Управление данными**: пользователи, категории, платежи
- **Диаграммы** для визуализации платежей
- **Экспорт** в Word и Excel
- **Смена пароля** с хешированием SHA1

## Технологии
- WPF (.NET Framework)
- Entity Framework
- SQL Server
- Microsoft Office Interop (Word/Excel)
- Windows Forms Data Visualization

## База данных
Таблицы: `User`, `Category`, `Payment`  
Связи: пользователь → платежи ← категории

## Особенности
- Валидация ввода данных
- Всплывающие подсказки
- Подтверждение действий
- Словари стилей
- Система контроля версий
